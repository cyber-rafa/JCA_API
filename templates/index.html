Como funciona uma API

Abra o Postman para fazer as requisições

GET: Busca informações no servidor. Exemplo: retornar todos os alunos → /alunos.

POST: Envia informações para o servidor. Exemplo: cadastrar um novo aluno → /alunos com corpo JSON.

PUT: Atualiza uma informação existente. Exemplo: atualizar dados do aluno → /alunos/id com JSON atualizado.

DELETE: Remove uma informação no servidor. Exemplo: deletar aluno → /alunos/id.
